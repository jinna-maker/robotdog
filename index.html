RoboDog v1

A 3D printed robot dog with ESP32, servo motors, OLED face display, and MQ135 gas/smoke sensing.

Story Parts Hardware Build Print Reminder Safety

Project Story

RoboDog v1 was created by Jinna Arunplod as a maker robotics project. The idea came from loving dogs but not being able to keep a real dog easily because of dog-fur allergy. This project turns that personal idea into a working robot using 3D printing, electronics, coding, and creativity.

Main Features

ESP32 controller
MG90S servo movement
OLED robot face
MQ135 gas/smoke sensing
Stand, sit, sleep, and give hand
Designed for STEM learning

How to Build

1. Print the 3D Parts

Print the body + head, four upper legs, four lower legs, four feet, and two ears.

2. Center the Servos

Before assembly, connect each servo and move it to 90 degrees. Attach the servo horn only after the servo is centered.

3. Assemble the Robot

  1. Install the hip servos into the body.
  2. Attach upper leg parts to the hip servos.
  3. Attach lower leg parts and feet.
  4. Install the head, OLED display, and ears.
  5. Mount the MQ135 sensor in a place with airflow.

4. Wire the Electronics

Connect the servos, OLED display, and MQ135 sensor according to the pin table. Use a separate 5V power supply for the servos.

5. Upload Code

  1. Open Arduino IDE.
  2. Install ESP32 board support.
  3. Install required libraries.
  4. Open the RoboDog code.
  5. Check the GPIO pin numbers.
  6. Upload to ESP32.

6. Test Movements

Test stand, sit, sleep, and give hand one at a time. Adjust servo angles if any leg moves in the wrong direction.

Important Printing Reminder

Do not print all files immediately. First print one upper leg, one lower leg, and one foot. Test the fit with the MG90S servo motor. After confirming the fit and movement clearance, print the remaining parts.

Safety Notes

  • Do not power all servo motors from the ESP32 USB port.
  • Use a separate 5V power supply for servo motors.
  • Connect all grounds together.
  • MQ135 is for educational demonstration only, not for life-safety gas detection.
  • Adult supervision is recommended for young makers.
